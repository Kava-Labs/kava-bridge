# Kava Etherium Bridge

A contract for cross-chain ERC20 token transfers.

## Setup

```
npm install
```

## Test

```
npm test
```

## Development

```
# Watch contract + tests
npm run dev

# Watch tests only
npm run test-watch
```

## Deploy with Hardhat
```
KAVA_BRIDGE_RELAYER_ADDRESS=0x8ba1f109551bd432803012645ac136ddd64dba72 npx hardhat run scripts/deploy.ts
```
